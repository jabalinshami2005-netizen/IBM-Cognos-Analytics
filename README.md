# IBM-Cognos-Analytics
“A healthcare analytics dashboard project built using IBM Cognos Analytics and web technologies to monitor hospital performance metrics.”

MediTrack Pro – Healthcare Analytics Dashboard

Project Overview

This project focuses on analyzing hospital operations using data-driven techniques and visualizing key insights through both IBM Cognos Analytics and a web-based dashboard. The goal is to monitor hospital performance, improve resource utilization, and enhance patient care quality.

The project is divided into two main parts:

* **MedCare Analytics (IBM Cognos)** – Business intelligence reports, dashboards, and insights
* **MediTrack Pro (Web Application)** – Interactive website for visualizing hospital data

MedCare Analytics (IBM Cognos)

Using IBM Cognos Analytics, multiple reports and dashboards were created to analyze key healthcare metrics:

Key Features

* Patient-level reporting and grouped summaries
* Bed Occupancy Rate analysis by ward
* Wait Time and Length of Stay insights
* Patient Satisfaction analysis
* Re-admission tracking

Advanced Analysis

* Conditional formatting to highlight critical wards (>90% occupancy)
* Exploration of relationships between wait time and satisfaction
* Identification of high-risk departments

MediTrack Pro (Web Dashboard)

A web-based dashboard was developed to provide an interactive and user-friendly interface for hospital analytics.

 Features

* Interactive charts (Bar, Line, Pie)
* Bed occupancy visualization by ward
* KPI cards (Total Patients, Critical Cases, Recovered)
* Dynamic chart switching
* Summary of top-performing and critical wards

Technologies Used

* IBM Cognos Analytics
* HTML, CSS, JavaScript
* Chart.js
* Replit (AI-assisted development)
* GitHub (version control & hosting)

Key Insights

* High bed occupancy leads to increased patient wait times
* Departments with heavy load show lower satisfaction levels
* Resource optimization can significantly improve hospital performance

Conclusion

This project demonstrates how data analytics and visualization can support healthcare management in making informed, data-driven decisions. By integrating business intelligence tools with modern web technologies, the system provides a comprehensive solution for monitoring and improving hospital operations.
